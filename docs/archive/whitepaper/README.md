# Whitepaper Support Files

This directory contains reference materials used during whitepaper creation.

## Files

- **WHITEPAPER_VISUALS.md** - Mermaid diagram source code (12 diagrams)
- **WHITEPAPER_DIAGRAMS_ASCII.md** - ASCII art diagrams (10 diagrams)
- **WHITEPAPER_README.md** - Distribution guide and regeneration instructions

## Usage

These files are kept for reference and future updates. To regenerate the whitepaper PDF:

```bash
cd /path/to/gpu-scatter-gather
./scripts/generate_whitepaper_pdf.sh
```

The main whitepaper source is at: `docs/WHITEPAPER.md`
The final PDF is at: `docs/GPU_Scatter_Gather_Whitepaper_v1.0.0.pdf`
